SC LOADOUT OPTIMIZER — Alpha 4.8.0-LIVE
========================================

Star Citizen ship loadout optimizer with damage calculator,
shop finder, and matchup analysis.

REQUIREMENTS
------------
- Python 3.x (https://www.python.org/downloads/)
  Make sure "Add to PATH" is checked during install.
- Any modern web browser (Chrome, Firefox, Edge)

HOW TO START
------------
Windows:  Double-click start.bat
Mac/Linux: Run ./start.sh in terminal

The app opens automatically in your browser at http://localhost:8765

HOW TO USE
----------
1. Select a ship from the list on the left
2. Choose PvE, PvP, or both
3. Click "Optimize Loadout"
4. Use dropdowns to swap weapons/components
5. Check the shopping list for where to buy everything
6. Select a target ship at the bottom for damage matchup analysis
7. Save loadouts for later

DATA
----
All data from api.star-citizen.wiki (game file extraction).
Patch: Alpha 4.8.0-LIVE (Build 11875683)
Data date: 2026-05-24
